# Windows Terminal

## Tab management
- Ctrl+Shift+T | New tab (default profile) | Opens a new tab using your default profile (e.g., PowerShell or Ubuntu).
- Ctrl+Shift+D | Duplicate tab | Opens a new tab with the same profile as the current tab.
- Ctrl+Shift+W | Close tab | Closes the currently focused tab.
- Ctrl+Tab | Next tab | Cycles to the next tab to the right.
- Ctrl+Shift+Tab | Previous tab | Cycles to the previous tab to the left.
- Ctrl+Alt+Number (1–9) | Jump to tab N | Switches directly to a specific tab by index.

## Pane management
- Alt+Shift+Plus (+) | Split pane vertically | Splits the current tab into side-by-side panes.
- Alt+Shift+Minus (-) | Split pane horizontally | Splits the current tab into top/bottom panes.
- Ctrl+Shift+W | Close pane | When a pane is focused, closes that pane (same shortcut as closing a tab).
- Pane navigation / focus movement
- Alt+Arrow Keys (← ↑ ↓ →) | Move focus between panes | Changes focus to the adjacent pane in the arrow direction.

## Command palette / actions
- Ctrl+Shift+P | Open command palette | Shows a searchable list of all actions; you can execute actions or discover their shortcuts.
- Copy / paste / search / text view
- Ctrl+Shift+C | Copy | Copies selected terminal text to the clipboard.
- Ctrl+Shift+V | Paste | Pastes clipboard text into the terminal.
- Ctrl+Shift+F | Find in buffer | Opens search box to find text in the scrollback buffer.

## Settings / profiles / window mode
- Ctrl+, | Open settings | Opens Windows Terminal settings UI.
- Alt+Shift+F | Toggle focus mode | Hides tabs and UI chrome to focus on terminal content.
