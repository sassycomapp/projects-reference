# Ubuntu

## System / launcher access
- Super | Open Activities / overview | Shows workspaces, open windows, and search; starting point to launch apps or switch context.
- Super + A | Application overview | Opens the grid of installed applications (app launcher).
- Super + S | Workspace overview | Shows workspaces and windows without opening the app grid.

## Application switching
- Alt + Tab | Switch applications | Cycles through open applications; keep holding Alt and tap Tab to move, release to select.
- Super + Tab | Switch workspaces/apps | On some Ubuntu layouts, cycles between workspaces and focused apps.
- Alt + Esc | Next window | Steps through windows in the current workspace in order.

## Window management
- Super + ↑ | Maximize window | Maximizes the currently focused window.
- Super + ↓ | Restore / minimize | Restores a maximized window or minimizes it if already restored.
- Super + ← | Tile left | Snaps the current window to the left half of the screen.
- Super + → | Tile right | Snaps the current window to the right half of the screen.
- Alt + F4 | Close window | Closes the focused application window.

## Workspace management
- Super + Page Up | Move to workspace above | Switches to the workspace above the current one.
- Super + Page Down | Move to workspace below | Switches to the workspace below the current one.
- Shift + Super + Page Up | Move window to workspace above | Sends the current window to the workspace above.
- Shift + Super + Page Down | Move window to workspace below | Sends the current window to the workspace below.

## Screenshots / notifications / lock / power
- PrtSc | Screenshot (full screen) | Takes a screenshot of the entire screen using Ubuntu’s screenshot tool.
- Shift + PrtSc | Area screenshot | Lets you select a rectangular region to capture.
- Ctrl + Alt + L | Lock screen | Immediately locks the session (shows lock screen).
- Super + V | Notification list | Opens the notification list/indicator (on current GNOME-based Ubuntu).

## Terminal launch / run command
- Ctrl + Alt + T | Open terminal | Launches the default terminal window (usually GNOME Terminal).
- Alt + F2 | Run command dialog | Opens a small run box to execute a command by name (e.g., gnome-terminal).
