# DOC-INDEX.md template

Regenerated each time `/align-docs` runs. Not hand-maintained — treat the previous version as disposable.

One section per document in scope:

```markdown
## <document filename>

**Defined terms:** term — short definition (flag if not already in GLOSSARY.md)

**Claims:**
- <component/requirement/decision stated in this document, in one line>
- ...

**References out:** <other documents, sections, or components this document points to>
```

Keep each claim short enough to compare at a glance against another document's claims — this file exists to make Step 3 (cross-reference) mechanical, not to re-explain the document.
